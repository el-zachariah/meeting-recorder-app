# Meeting Recorder v0.7.1 release signoff

Date: 2026-06-05
Commit: pending tag at release time

## Product shape

- `meeting-recorder gui` launches the modern WebView-style recorder UI path.
- `meeting-recorder gui-screenshot` renders the modern recorder panel with visible recording controls, setup/status information, and save/transcribe workflow evidence.
- v0.7.1 specifically fixes the modern UI action bridge so frontend buttons are not static-only visuals.
