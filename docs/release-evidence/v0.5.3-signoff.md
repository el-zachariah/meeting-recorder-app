## Known limitations to carry into release notes

- Normal GUI startup requires a desktop tray/AppIndicator stack (`python3-pystray`, Pillow, `python3-gi`, and Ayatana/AppIndicator GIR bindings) plus a desktop environment that displays tray icons.
- The screenshot command renders the Tk dropdown evidence surface without the native tray icon so it can run in CI/installed-artifact smoke tests.
- Screen video capture remains X11/XWayland-oriented; audio-first recording remains the default.
